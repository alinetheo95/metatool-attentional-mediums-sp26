HTML Crash Course

A hands-on introduction to building web pages

How to use this course

Each lesson is an HTML file you can open in your browser. You'll learn by reading, editing, and refreshing:

  1. Open the HTML file in a text editor (like VS Code)
  2. Open the same file in a web browser
  3. Make changes in the editor, save, and refresh the browser to see results
Tip: Right-click on any webpage and select "Inspect" to open developer tools. This lets you see the HTML and CSS behind any page.

Lessons

  1. Hello World
    Your first webpage. Learn the basic structure of an HTML file.
  2. Intro to HTML
    Understand elements and tags -the building blocks of HTML.
  3. HTML Elements
    Explore common elements: headings, paragraphs, divs, images, links, and lists.
  4. Intro to CSS
    Style your pages with colors, borders, spacing, and more.
  5. Adding Fonts
    Make your page unique with custom fonts from Google Fonts.
  6. Index Template
    A starter template for your own documentation page. Copy it and make it yours.

When you're ready

After completing the lessons, copy 6_index_template.html to your own folder, rename it to index.html, and start documenting your work.
