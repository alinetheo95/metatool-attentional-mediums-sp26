Adding Google Fonts

So far, your page is using the browser's default font. Google Fonts offers hundreds of free fonts you can add to any webpage.

How it works

Adding a Google Font requires two steps:

  1. Link to the font — Add a special <link> tag in your <head> that loads the font from Google's servers.
  2. Use it in CSS — Add font-family to your CSS rules to apply the font.

Step-by-step

  1. Go to fonts.google.com and browse for a font you like.
  2. Click on a font, then click "Get font" and "Get embed code".
  3. Copy the <link> code and paste it into the <head> section of this file (above the <style> tag).
  4. Google will also show you CSS like this: 
    font-family: "Roboto", sans-serif;
    Add that line to any CSS class you want to style. For example, to change the title font, add it to .title.

Observation on typography and readability
Typography affects how people read and feel about content. A well-chosen font can make text easier to read, convey personality, and create visual hierarchy. Try different fonts and notice how the same words feel different.
